Task 2. Практическое задание:
различия методов get и load в способе загрузки данных из базы. Метод load выполняет ленивую загрузку,
 потому если объект в базе не найден (попытка загрузки несуществуеющего идентификатора), результатом работы метода будет Exception
Результатом работы метода get при попытке загрузки несуещствующего объекта из базы данных будет null
